# 💼 Portfólio Dev - Walker Pontes

Este é o meu site de portfólio pessoal, desenvolvido para apresentar meus projetos, habilidades e experiências na área de desenvolvimento de software.

## 🚀 Sobre o projeto

O objetivo deste portfólio é centralizar minhas principais criações como desenvolvedor, destacando minhas habilidades em desenvolvimento web, APIs REST e construção de interfaces modernas e intuitivas.

Além disso, o projeto também serve como vitrine profissional para recrutadores e clientes.

## 🛠️ Tecnologias utilizadas

- ⚛️ React (Vite)
- 🎨 Tailwind CSS
- 🎬 Framer Motion
- 🧩 shadcn/ui
- 🌐 HTML5 & CSS3
- 📜 JavaScript / TypeScript

## ✨ Funcionalidades

- Apresentação pessoal
- Listagem de projetos com links
- Certificados e formações
- Layout responsivo (mobile e desktop)
- Animações suaves com Framer Motion
- Tema moderno (dark/light mode)
